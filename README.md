# Partner Portal

## Overview
Partner Portal is a web dashboard that helps partner users view updates, join discussions, and communicate through built-in email tools.

## Navigation
The portal includes the following main sections:

- **Home**
  - Landing page after login. Provides quick access to the portal features.

- **Announcements**
  - **Exclusive Announcements**: Partner-only updates.
  - **General Announcements**: Updates shared with all partners.

- **Discussion**
  - **Open New Topic**: Create a new discussion thread.
  - **View Discussion**: Browse and read existing discussion topics.

- **Email**
  - Email section for partner communication (for example, contacting the portal/admin team).

- **LogOut**
  - Signs the user out and ends the current session.

## Typical Flow
1. Open **Home**
2. Check **Announcements**
3. Join or start a **Discussion**
4. Use **Email** if needed
5. Click **LogOut** to finish
